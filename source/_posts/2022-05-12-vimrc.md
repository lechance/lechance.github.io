##### vimrc

    colorscheme elflord
    syntax on
    set laststatus=2
    set tabstop=2
    set shiftwidth=2
    set autoindent
    set noexpandtab
    set expandtab
    set softtabstop=2
    set autowrite
    set number

##### vim

`:set mouse=a` to enable scrolling with the scroll wheel, as well as selecting to enter Visual mode, etc.

`:set mouse=n` if you only want scrolling enabled in normal mode.
